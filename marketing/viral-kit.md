# 饭桶研究所传播素材包

目标：公开首发先打“饭点人格测试”。让用户觉得“这说的不就是我吗”，再让他愿意截图发群。

## 首发主卖点

```text
不是 MBTI，是你的饭点行为识别报告。
```

副卖点：

```text
研究所内参式报告 / 一键生成海报 / 顺手决定今天吃啥
```

## 3 条首发笔记

### 1. 打工人午饭人格

标题：

```text
测了一下我的饭点人格，离谱但准确
```

正文：

```text
发现一个很适合午休摸鱼的小测试：饭桶研究所的饭点人格测试。

它不测 MBTI，只测你吃饭时暴露出来的真实状态，还会生成一份很像研究所内参的饭点行为识别报告。

我测出来是「满减精算型」：
满减不是优惠，是我和平台之间的智力对局。

真的有被冒犯到，但我承认。

你们测出来是什么？把报告海报发出来我看看谁最像饭点危险样本。
```

标签：

```text
#饭桶研究所 #饭点人格 #打工人午饭 #外卖满减 #搞笑测试
```

### 2. 夜宵人格

标题：

```text
夜里十一点以后测这个，准确率有点可怕
```

正文：

```text
白天可以假装自律，晚上不能骗胃。

测了一下饭桶研究所，我居然是「夜宵复活型」：
白天是打工人，晚上是夜宵复活甲持有人。

它还给我生成了饭桶行为识别报告，荒诞得很正式。

建议夜宵局开吃前先测，测完再决定谁负责点烧烤。
```

标签：

```text
#夜宵 #饭点人格 #搞笑测试 #打工人日常 #今天吃什么
```

### 3. 群聊挑战

标题：

```text
把这个发群里，立刻识别谁最会点饭
```

正文：

```text
今天发现一个适合发群里的饭点测试。

它会把你识别成 16 种饭桶行为类型，比如：
满减精算型、吃啥纠结型、拼单外交型、夜宵复活型。

最适合午饭前发群：
“都测一下，我看看今天谁负责决定吃啥。”

我愿称之为群聊饭点责任分配系统。
```

标签：

```text
#群聊日常 #饭点人格 #今天吃啥 #搞笑测试 #饭桶研究所
```
