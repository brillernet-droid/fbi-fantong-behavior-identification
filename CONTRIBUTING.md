# Contributing

欢迎给饭桶研究所投喂新想法。

## Copywriting Rules

- 保持幽默、荒诞、meme-friendly
- 不要低俗、攻击个人或冒犯特定群体
- `FBI` 只能解释为 `FanTong Behavior Identification`
- 不要使用真实 FBI 或任何真实政府、执法机构品牌元素
- 不要使用徽章、印章、鹰形标识、警徽、制服等官方机构拟态视觉

## Local Check

```bash
npm run check
```

如果只是改 HTML/CSS/JS，可以直接打开 `index.html` 或运行：

```bash
npm start
```
